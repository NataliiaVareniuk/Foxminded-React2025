# React Todo App

Responsive todo application built with React and Vite.
Demonstrates CRUD operations, server state management with React Query, and modular SCSS architecture.

**Demo**
[View the live app](https://react-todo-app-vn.netlify.app/)

## Key Features

- Full CRUD (create, read, update, delete) todo items
- Server state synchronization via TanStack React Query
- REST API simulation using json-server
- Optimistic updates and cache invalidation
- Modular SCSS styling
- Responsive layout

## Local Setup

Install dependencies:
npm install

Start API server:
npm run dev:server

Start client:
npm run dev:client

Build for production:
npm run build

This starts the Vite development server, typically on http://localhost:5173.
