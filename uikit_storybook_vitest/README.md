# UIKit — React Component Library

Reusable React UI component library with isolated development and documentation via Storybook.

**Live Demo**

**Application:**  
[View Live App](https://projectsreactuikit.netlify.app/)

**Storybook:**  
[View Storybook](https://reactstorybook.netlify.app)

**Overview**
The project demonstrates a modular UI library architecture with documented and unit-tested components. Each component is developed in isolation and validated with automated tests.

**Development**

**Install dependencies:**
npm install

**Run dev server:**
npm run dev

**Run Storybook:**
npm run storybook

**Run tests:**
npm run test

**Build:**
npm run build
